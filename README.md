# Cambridge-Colleges-TSP
Travelling Salesman Problem solved for Colleges around Cambridge. Route optimised via a Genetic Algorithm.

#### Usage
Setup Google Maps API service to get an API key

Rename .env_sample to .env and enter the API key 

Data is already provided for geocodes, distances and directions between colleges but can be updated by running any of the respective files, e.g to update geocodes run geocode.py

Run the solver with solver.py
